# Add command bar/palette for quick actions

## Feature Request: Command Bar/Palette

Add a VS Code/Cursor-style command bar for quick access to common actions.

### Key Requirements
- **Activation**: Meta+P or Ctrl+P keyboard shortcut
- **Core functionality**: Quick access to actions like "Authenticate new Google account"
- **Integration**: Work with existing TUI framework (@opentui/core, @opentui/solid)

### Architecture Reference
Research SST OpenCode's pluggable command system for implementation inspiration.

### Priority
Nice-to-have UX improvement to enhance user experience and discoverability.
